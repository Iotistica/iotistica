# API Overview

The Iotistica API is used for device lifecycle, target-state control, and telemetry integration.

## Typical Workflows

- Register and provision devices
- Push target state updates
- Query current runtime state
- Retrieve usage and health metrics

## Security Model

- JWT-based authentication for platform access
- License validation for feature gating
- Scoped credentials for service integrations
