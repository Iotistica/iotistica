# Quickstart

This quickstart gets a first Iotistica edge agent connected and visible.

## Prerequisites

- Linux edge device (Raspberry Pi or x86-64)
- Outbound internet access
- Docker installed on the target device

## Install the Agent

```bash
curl -sfL https://get.iotistica.com/agent/install | sh
```

## Verify Service Health

```bash
curl http://localhost:48484/v2/device
```

A healthy response should include the device UUID and current status.
